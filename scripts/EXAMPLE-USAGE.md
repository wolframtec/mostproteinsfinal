# NanoGPT Video Generation - Quick Examples

## Your Use Case: DNA Background Loop

### 1. Generate Your Loop

```bash
# Set your API key
export NANOGPT_API_KEY="your-api-key-here"

# Generate seamless loop for desktop (16:9)
npm run video:generate -- "DNA double helix rotating slowly in dark biotech laboratory, microscopic view, subtle cyan glow, cinematic lighting" --duration 10 --loop --output dna-desktop

# Generate for mobile (9:16)
npm run video:generate -- "DNA strands forming helix structure, dark background, bioluminescent particles floating, seamless rotation" --aspect 9:16 --duration 8 --loop --output dna-mobile
```

### 2. Use the Automated Script (Recommended)

```bash
# This generates AND splits for zoom-then-loop pattern
npm run video:loop -- "DNA helix rotating in dark void with bioluminescent particles"

# With options
npm run video:loop -- "Abstract biotech cells dividing" --duration 10 --aspect 16:9
```

### 3. Manual Workflow

```bash
# Step 1: Generate base video
node scripts/generate-video-loop.js "DNA helix dark background" --duration 10 --output my-dna

# Step 2: Extend it for seamless loop
node scripts/generate-video-loop.js "DNA helix continuation" --extend --output my-dna-extended

# Step 3: Split using ffmpeg
ffmpeg -i public/videos/my-dna-extended.mp4 -t 5 -c copy zoom.mp4
ffmpeg -ss 00:00:05 -i public/videos/my-dna-extended.mp4 -c copy closeup.mp4

# Step 4: Deploy
npm run build && npm run deploy
```

## Prompt Engineering Tips

### For Seamless Loops

**Good prompts:**
- "DNA helix rotating continuously"
- "Abstract particles floating in space, no camera movement"
- "Bioluminescent cells pulsing rhythmically"
- "Microscopic organisms drifting slowly"

**Avoid:**
- "Camera panning left to right" → creates visible cut
- "Object enters from left, exits right" → doesn't loop
- "Transition from A to B" → one-way animation

### Style Keywords That Work Well

- **Lighting:** "cinematic lighting", "volumetric fog", "bioluminescent glow"
- **Color:** "cyan and teal", "dark blue and green", "monochrome"
- **Mood:** "futuristic", "sci-fi", "laboratory", "microscopic"
- **Quality:** "high detail", "sharp focus", "smooth motion"

### Example Prompts for Most Proteins

```bash
# DNA Theme
"Rotating DNA double helix structure, dark biotech laboratory background, subtle blue-green bioluminescent glow, microscopic view, particles floating, cinematic lighting, high detail"

# Abstract Biotech
"Abstract organic cells dividing and multiplying, dark background, teal and cyan color palette, microscopic view, smooth fluid motion, seamless loop, biotech aesthetic"

# Laboratory
"Futuristic biotech laboratory with holographic DNA displays floating, dark aesthetic, subtle green accent lighting, sci-fi environment, depth of field"

# Microscopic
"Microscopic view of protein chains folding, dark background, realistic biological structures, subtle movement, scientific visualization style"
```

## Troubleshooting

### "NANOGPT_API_KEY not set"
```bash
export NANOGPT_API_KEY="your-key"
```

### "Generation failed"
- Check account balance: https://nano-gpt.com/dashboard
- Ensure prompt doesn't violate policies
- Try shorter prompt

### "Video doesn't loop seamlessly"
- Use `--loop` flag
- Try prompts with continuous motion
- Use `--extend` feature
- Check that first and last frames match visually

### "ffmpeg not found" (when using create-seamless-loop.sh)
```bash
# macOS
brew install ffmpeg

# Ubuntu/Debian
sudo apt install ffmpeg

# Windows
# Download from https://ffmpeg.org/download.html
```
